# [Ely Has Pride Website](https://elyhaspride.github.io/website/)

[![LICENSE](https://img.shields.io/badge/license-MIT-lightgrey.svg)](https://raw.githubusercontent.com/elyhaspride/website/master/LICENSE)
[![Jekyll](https://img.shields.io/badge/jekyll-%3E%3D%203.7-blue.svg)](https://jekyllrb.com/)

> No need to run anything on your computer.
> Just edit pages on GitHub and the website updates automatically.

## Edit the website

Most contributors only need to edit pages in the `_pages/` folder.
Other folders support the website framework and usually do not need changes.

## Spelling

This repository uses a local spelling dictionary (`cspell.json`)
to account for names, acronyms, and local terminology.
